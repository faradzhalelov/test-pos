# pos_order
ТЕСТОВОЕ ЗАДАНИЕ
Сделать приложение с SQLite базой внутри приложения, с работой в офлайн режиме.
Наполнить можно БД вручную
Группа товаров 1
Товар 1.1
Товар 1.2
Группа товаров 2
Товар 2.1
Товар 2.2
Товар 2.3
И т.д.
Возможность набирать заказы, сохранять и отобразить список сохраненных заказов.
